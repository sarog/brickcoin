# BrickCoin

## Overview

This handy little browser extension allows you to quickly compare BrickLink and LEGO.com Pick-A-Brick item prices with the goal of saving money.

## Install the extension

- Load the extension into your browser of choice using developer mode. Detailed instructions for Chrome and Firefox to be provided at a later date.
- In the extension’s options, select the country and language you normally use on the official LEGO website.

## Using the extension

With BrickCoin installed, the next step is to ensure you have a browser tab open for LEGO.com with your LEGO account logged in.
Ideally, only keep one LEGO.com tab open at a time as the extension looks for the first tab with the 'lego.com' URL in the browser.
Keeping multiple tabs open across many browser windows will create a messy experience.

Start creating one or more carts on BrickLink as you normally do. For performance reasons, start small and keep the total lot count under 500.
Once done, visit [your global cart](https://www.bricklink.com/v2/globalcart.page) on BrickLink and wait for the extension to work its magic.
After a brief loading period, the extension will display potential savings by comparing item prices between LEGO’s Pick-A-Brick (PAB) and BrickLink.
